# Pere Pages Soms CV

## Howto

My curriculum vitae in REACT and Angular1.5.

Printable and reponsive.

### Install

```bash
mpm i
```

### Run

```bash
# runs webpack-dev-server
npm run dev
```

or

```bash
# builds and runs
npm start
```
